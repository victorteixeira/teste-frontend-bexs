# Teste front-end Bexs

## Como usar

Antes de usar, instale o Node.js v12.13.1 ou maior.

#### Tecnologias usadas:
* [React](https://reactjs.org/)
* [React Hooks](https://pt-br.reactjs.org/docs/hooks-intro.html)
* [Atomic Design Methodology](http://atomicdesign.bradfrost.com/)
* [SASS](https://sass-lang.com/) with syntax SASS

#### Para rodar:

```
npm start
```

A aplicação irá rodar em http://localhost:3000
